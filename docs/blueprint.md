# **App Name**: BlancoMarket

## Core Features:

- Product Catalog Browsing: Display all available products with high-quality images, names, and prices in a grid format.
- Filtering & Sorting: Allow users to filter products by categories, price range, and availability, as well as sort results by different criteria.
- Individual Product Pages: Provide a dedicated page for each product with detailed descriptions, multiple images, and product-specific information.
- User Authentication & Profiles: Enable users to register, log in, manage their account details, and view their order history or wishlist.
- Wishlist Functionality: Allow logged-in users to save products to a personal wishlist for future reference.
- Admin Product Management: A secure admin interface for adding new products, editing existing listings, updating inventory, and marking products as 'Nuevo', 'Oferta', or 'Agotado'.
- AI Product Description Assistant: An AI tool for administrators to generate creative and descriptive product texts based on keywords and brief inputs.

## Style Guidelines:

- Primary color: A sophisticated, muted taupe-brown (#5C554F) to anchor content, evoking natural comfort and warmth.
- Background color: A soft, creamy off-white (#F2F1EF) providing a clean, bright canvas.
- Accent color: A gentle, elegant rose-pink (#ECB3C4) used for interactive elements, buttons, and status indicators like 'Nuevo' or 'Oferta', adding a touch of softness.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral aesthetic, ensuring excellent readability across all screen sizes and content types.
- Utilize a set of modern, minimalist outline icons for navigation, filters, and user actions to maintain a clean and uncluttered interface.
- Implement a clean and spacious grid layout for product display, accompanied by an intuitive sidebar for filtering and navigation. Maintain ample white space for a premium feel.
- Incorporate subtle hover effects on product cards and smooth transitions for filtering actions to enhance user interaction without distraction.
